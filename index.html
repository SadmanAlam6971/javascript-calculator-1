JavaScript Basic Calculator

Simple UI for your assignment — use the controls below and click Calculate.

Type two numbers on the left, choose an operation and press Calculate. The calculator calls the JavaScript function defined in your cal file so results will match your assignment logic and error messages.

  • Division by zero will show: Error: Division by zero is not allowed
  • Invalid operation will show: Error: Invalid operation
  • Non-number input will show: Error: Invalid number input
